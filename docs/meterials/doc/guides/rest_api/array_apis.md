# Array APIs
## CREATEARRAY
| REST element |                Value                |
|:------------:|:-----------------------------------:|
| Method       | POST                                |
| Type         | RAW                                 |
| URL          | http://{{host}}/api/ibofos/v1/array |

### REST body in the request
```
{
    "param": {
        "name": "{{arrayName}}",
        "raidtype": "RAID5",
        "buffer": [
            {
                "deviceName": "uram0"
            }
        ],
        "data": [
            {
                "deviceName": "{{deviceName1}}"
            },
            {
                "deviceName": "{{deviceName2}}"
            },
            {
                "deviceName": "{{deviceName3}}"
            }
        ],
        "spare": [
            {
                "deviceName": "{{deviceName4}}"
            }
        ]
    }
}
```
